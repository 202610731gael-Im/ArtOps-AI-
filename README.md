# ArtOps AI  
문화예술·행사 기획·운영 자동화 플랫폼

## 1. 프로젝트 소개

**ArtOps AI**는 공연, 문화예술 행사, 연주회, 페스티벌, 기획 공연 등에서 필요한  
기획서 작성, 보고서 작성, 홍보물 초안 생성, 공연 계약서 작성 및 PDF 변환까지 지원하는  
**AI 기반 문화예술 운영 자동화 플랫폼**입니다.

공연예술 현장에서 반복적으로 발생하는 문서 작성 업무를 줄이고,  
기획자·운영자·연주자·단체가 더 빠르고 정확하게 실무를 처리할 수 있도록 돕는 것을 목표로 합니다.

---

## 2. 핵심 문제 정의

문화예술 및 공연 행사 운영 과정에서는 다음과 같은 문제가 반복됩니다.

- 공연 기획서, 결과보고서, 홍보문구 작성에 많은 시간이 소요됨
- 계약서 작성 시 이름, 악기 파트, 생년월일, 연락처, 계좌번호 등 반복 입력이 필요함
- 연주자별 계약서를 수작업으로 작성하고 PDF로 변환하는 과정이 번거로움
- 서명 및 도장 처리 후 문서 보관과 재활용이 어렵고 비효율적임
- 소규모 예술단체나 기획사는 전담 행정 인력이 부족함

ArtOps AI는 이러한 반복 행정 업무를 자동화하여  
공연예술 실무자의 업무 시간을 줄이고 문서 품질을 높이는 것을 목표로 합니다.

## 4. 핵심 기능

## 4.1 AI 문서 초안 생성

공연 및 행사 정보를 입력하면 AI가 다양한 문서를 자동으로 초안 작성합니다.

### 지원 문서 예시

- 공연 기획서
- 행사 제안서
- 결과보고서
- 보도자료
- SNS 홍보문구
- 포스터 문구
- 관객 안내문
- 후원 제안서
- 사업계획서 초안

### 입력 예시

- 공연명
- 공연 일시
- 장소
- 출연진
- 프로그램
- 기획 의도
- 대상 관객
- 예산 규모
- 주최/주관 정보

---

## 4.2 공연예술 계약서 자동 생성

연주자 정보와 공연 정보를 입력하면 공연 계약서를 자동으로 생성합니다.

### 저장 가능한 연주자 DB 정보

- 이름
- 악기 파트
- 생년월일
- 전화번호
- 통장 계좌번호
- 이메일
- 주소
- 소속
- 비고

### 계약서 자동 생성 흐름

1. 연주자 선택
2. 공연 정보 입력
3. 계약 조건 입력
4. 계약서 자동 작성
5. 서명 및 도장 적용
6. PDF 파일 생성
7. 다운로드 및 보관

- 계약서 미리보기
- 연주자별 계약 이력 관리

## 5. MVP 기능 범위

초기 버전에서는 다음 기능을 우선 개발합니다.

### 1차 MVP 필수 기능

1. **연주자 DB 관리**
   - 연주자 등록
   - 연주자 목록 조회
   - 연주자 정보 수정 및 삭제
   - 이름, 악기 파트, 생년월일, 전화번호, 계좌번호 저장

2. **공연 계약서 자동 생성**
   - 연주자 선택
   - 공연명, 일시, 장소, 출연료 입력
   - 계약서 템플릿 기반 자동 작성
   - 계약서 미리보기

3. **PDF 변환 및 다운로드**
   - 완성된 계약서를 PDF로 변환
   - 서명 및 도장 이미지 적용
   - 파일 다운로드

### 2차 확장 기능

- AI 기획서 작성
- AI 결과보고서 작성
- 홍보물 문구 생성
- 공연별 문서 관리
- 단체/기관별 템플릿 관리
- 계약서 일괄 생성
- 이메일 발송 기능
- 전자서명 연동

## 6. 서비스 사용 시나리오

### 시나리오 1: 연주자 계약서 작성

공연기획자는 공연명, 공연일, 장소, 출연료를 입력합니다.  
이후 등록된 연주자 DB에서 연주자를 선택하면 계약서가 자동으로 작성됩니다.  
사용자는 내용을 확인한 뒤 서명과 도장을 적용하고 PDF 파일로 다운로드합니다.

### 시나리오 2: 공연 기획서 초안 작성

사용자는 공연의 주제, 장소, 출연진, 프로그램, 기획 의도를 입력합니다.  
ArtOps AI는 이를 바탕으로 공연 기획서 초안을 생성합니다.  
사용자는 초안을 수정하여 제안서나 내부 문서로 활용할 수 있습니다.

### 시나리오 3: 홍보문구 생성

사용자는 공연 정보를 입력하고 홍보 목적을 선택합니다.  
예를 들어 SNS 홍보, 보도자료, 관객 안내문 등을 선택하면  
AI가 목적에 맞는 홍보 문구를 생성합니다.
